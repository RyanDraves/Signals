Detailed Description

template<typename Ret, typename... Args, typename FunctionType>
class sig::Slot< Ret(Args...), FunctionType >

Represents a function that can be connected to a Signal.

Slots can track other objects to mirror their lifetime. If one tracked object is destroyed, *this will no longer be called by any Signal. Slots can be called like any other function, with operator(). Is not nothrow move constructible/assignable because std::function is not.

Parameters
RetReturn type of the function.
Args...Argument types to the function.
FunctionTypeInternally held type where the function will be stored.
